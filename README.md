# Sistema de Gestão de Entregas

Projeto desenvolvido como Trabalho de Conclusão de Curso da Pós-graduação em Desenvolvimento Full Stack - PUCRS.

## Objetivo
Sistema web para gerenciamento de entregas, clientes e cálculo automático de valores por bairro.

## Tecnologias
- Node.js
- Express
- PostgreSQL
- Prisma ORM
- JWT
- Docker (em construção)
- GitHub Actions (em construção)

## Funcionalidades
- Autenticação de usuários
- Cadastro de clientes
- Tabela de preços por bairro
- Criação automática de pedidos
- Cálculo automático de valor da entrega

## Arquitetura
Backend estruturado em:
- Controllers
- Services
- Routes
- Middlewares

## Como executar
```bash
npm install
node server.js
